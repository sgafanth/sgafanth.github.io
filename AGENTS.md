# Agent Instructions

- Pull, commit, push.
- Create tests to reproduce bugs.
- Fix code to make tests pass.
- Always push your changes.
- Always push to feature branches, never directly to main. Create a PR for your feature branch.
- Never push to main, always create a PR and someone else will review it and merge it.
- Pre-commit hooks should perform formatting, linting, testing.
- GitHub Actions should run pre-commit hooks and deployment.
